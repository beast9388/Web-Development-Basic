Helmet Detection System

A web application for enhancing road safety by detecting persons on motorcycles or scooters and verifying helmet usage.

Key Features:
  • User Authentication: Secure sign up, log in, and log out.
  • Image Upload: Upload images for analysis.
  • Object Detection: Detects persons and helmets using YOLO models.
  • Visualization: Highlights detected elements with bounding boxes.
  • Feedback Collection: Users can submit feedback.
Technologies Used:
  • Backend: Flask, MySQL
  • Object Detection: OpenCV, YOLO models
  • Frontend: HTML, CSS, JavaScript

Role: Lead developer - implemented backend, database integration, YOLO model integration, frontend development, and security measures.

Noise Suppression System

The Noise Suppression System enhances audio quality by applying high-pass filtering to uploaded audio files, removing unwanted low-frequency noise.

Key Features:
  • User Authentication: Secure sign up, log in, and log out.
  • File Upload: Users can upload audio files for processing.
  • Audio Processing: High-pass filtering to remove noise.
  • Playback and Download: Processed audio files can be played back or downloaded.
  • Feedback Collection: Users can submit feedback.
Technologies Used:
  • Backend: Flask, MySQL
  • Audio Processing: Librosa, SciPy, SoundFile
  • Frontend: HTML, CSS, JavaScript

Role: Developed backend, implemented audio processing, designed frontend, and ensured functionality for file download and playback.

Mining Chatbot
